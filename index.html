Crop-Agent (Swift 1.2)

View on GitHub Download .zip Download .tar.gz

Crop Agent est une application iOS(8.0) universelle permettant de recadrer les images stockées dans "Photos". La photo et le format de sortie choisis, l'utilisateur peut editer l'image en modifiant sa taille, sa position ou son orientation. Les images ainsi éditées peuvent ensuite être imprimées ou simplement sauvegardées dans "Photos". 

> L'accès aux images et leur gestion est assurée via le framework "Photos". 

> L'affichage de l'image est pris en charge par un CATiledLayer pour minimiser son emprunte mémoire.
  Une version basse résolution de l'image est placée arrière plan pour fluidifier l'affichage.

> Les différents rendus graphiques sont effectués via "Core graphics".

> L'application a recours à "GCD" pour décharger le thread principal des taches les plus lourdes. 

> L'impression des photos est prise en charge par "AirPrint"
